О проекте

Данный учебный проект разработан с целью более полного погружения в удивительный кошачий мир. На просторах сети Интернет в настоящее время существует огромное количество информации о кошках, постараемся собрать на данном ресурсе наиболее интересные факты.

Демонстрируемый проект представляет собой Single Page Application (SPA), в рамках которого осуществлено подключение и взаимодействие с полноценной базой данных, где хранятся статьи о кошках.

При разработке данного одностраничного приложения использовались следующие инструменты: HTML5, CSS3, язык программирования JavaScript, библиотека Bootstrap, Node.js, для работы с базой данных - MySQL.

Приложение разработано в редакторе кода Visual Studio Code. Использовались следующие расширения для редактора кода: SQLTools, SQLTools MySql, LibMan Tools, JavaScript (ES6) code snippets. База данных разрабатывалась с помощью MySql Workbench.
